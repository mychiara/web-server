Dokumentasi Resmi

Panduan Instalasi Dashboard

Panduan langkah-demi-langkah dari nol untuk memasang dan menjalankan MasanDigital Dashboard pada server Linux (VPS, STB, Mini PC, atau Local Server).

Pengenalan

MasanDigital Dashboard adalah sistem panel web terintegrasi yang memudahkan pemantauan dan pengelolaan layanan server, container Docker, container LXD, website, status keamanan, serta konfigurasi jaringan melalui visualisasi antarmuka yang modern dan responsif.

Fitur Utama

Mendukung pengelolaan kontainer LXD (limit resource CPU/RAM), integrasi CasaOS otomatis, pemantauan status penyimpanan/disks, pengujian jaringan (speedtest), serta pengiriman notifikasi notifikasi bot Telegram/Discord ketika ada anomali sumber daya server.

1. Persiapan Sistem

Sebelum memulai proses instalasi, pastikan sistem operasi dan perangkat Anda memenuhi ketentuan berikut:

Sistem Operasi

Linux (Ubuntu 20.04+, Debian, Armbian, atau OpenWRT berbasis Docker).

Akses Administrator

Hak akses user non-root dengan instruksi sudo, atau login langsung sebagai root.

Koneksi Internet

Diperlukan untuk mengkloning kode repositori dan mendownload image Docker.

2. Metode Instalasi

Terdapat dua cara untuk menaruh file proyek di server target Anda:

A

Metode Git Clone (Sangat Direkomendasikan)

Metode ini paling efisien karena mempermudah pembaruan sistem di masa mendatang.

Bash 
# 1. Update list paket dan pasang git
sudo apt-get update && sudo apt-get install -y git

# 2. Clone repositori ke folder masandigital_dashboard
git clone https://github.com/mychiara/web-server.git masandigital_dashboard

# 3. Masuk ke folder proyek
cd masandigital_dashboard

# 4. Berikan izin eksekusi & jalankan installer
chmod +x install.sh
sudo bash install.sh
B

Metode File ZIP (Instalasi Manual)

Gunakan opsi ini apabila Anda memindahkan file manual (melalui WinSCP, FileZilla, dll) dalam bentuk paket file `.zip`:

Bash 
# 1. Install utilitas unzip jika belum tersedia
sudo apt-get update && sudo apt-get install -y unzip

# 2. Ekstrak file zip ke direktori masandigital_dashboard
unzip masandigital_dashboard_retail.zip -d masandigital_dashboard

# 3. Masuk ke direktori dan eksekusi script instalasi
cd masandigital_dashboard
chmod +x install.sh
sudo bash install.sh
Tugas Script Installer

Script install.sh akan otomatis mendeteksi CasaOS (memindahkan portnya ke 9999 agar tidak bentrok dengan port 80), memastikan port 80 kosong, memasang Docker (jika belum ada), serta membuild ulang kontainer dashboard.

3. Login Awal (Setup Wizard)

Setelah instalasi kontainer berhasil diselesaikan oleh script otomatis:

1

Buka web browser dan arahkan alamat ke IP server Anda (Contoh: http://192.168.1.10).

2

Sistem akan memandu Anda menuju halaman Setup Wizard untuk membuat kata sandi administrator pertama kali.

3

Masukkan kata sandi baru Anda, klik simpan, lalu gunakan kata sandi tersebut untuk proses login.

4. Pengelolaan Layanan Server

Berikut adalah beberapa perintah terminal penting yang sering digunakan untuk memantau, menghentikan, atau memperbarui container dashboard:

Perintah Docker Compose 
# Pastikan Anda berada di direktori instalasi:
cd /root/masandigital_dashboard

# Menghentikan sementara container dashboard
sudo docker compose down

# Menjalankan container di latar belakang (Background)
sudo docker compose up -d

# Melihat output logs real-time dashboard
sudo docker compose logs -f

# Update & Build ulang container setelah git pull
git pull origin main
sudo docker compose up -d --build

5. Troubleshooting (FAQ)

Klik pada pertanyaan untuk melihat solusi penyelesaian masalah:

Tidak. Script installer otomatis mendeteksi CasaOS dan memindahkan port layanannya dari port 80 ke port 9999. Anda tetap dapat membuka CasaOS via browser di http://IP_SERVER:9999.

1. Jalankan nano docker-compose.yml di dalam folder instalasi.
2. Cari baris port - "80:5000" lalu ubah menjadi - "PORT_PILIHAN_ANDA:5000" (contoh: `8080:5000`).
3. Simpan perubahan (Ctrl+X, tekan Y, tekan Enter).
4. Jalankan perintah sudo docker compose up -d untuk menerapkan port baru.

Hapus berkas proteksi konfigurasi keamanan lama dan restart kontainer untuk memunculkan kembali wizard set-up admin: 

cd /root/masandigital_dashboard
sudo rm data/security_config.json
sudo docker compose restart

Hal ini disebabkan port 80 masih diduduki oleh server lain (seperti Apache/Nginx bawaan OS host). Matikan dengan mengetik sudo systemctl stop apache2 atau sudo systemctl stop nginx, kemudian restart proses build container.

Kemungkinan browser menyimpan cache halaman yang usang. Lakukan hard refresh dengan menekan tombol kombinasi Ctrl + F5 (Windows) atau Cmd + Shift + R (macOS).
