Auto Clean RAM

Membersihkan cache memori RAM server secara otomatis setiap 6 jam.

0 */6 * * * sync; echo 3 > /proc/sys/vm/drop_caches

Bersihkan Log Lama

Mencari dan menghapus file log yang usianya di atas 7 hari setiap hari Minggu jam 2 pagi.

0 2 * * 0 find ... -delete
