Cara Menghubungkan iPhone (iOS)

Akses file dan backup foto Anda langsung dari aplikasi File iPhone bawaan.

1

Buka Aplikasi File (Files)

Aplikasi ini adalah bawaan dari Apple di iPhone Anda. Pastikan iPhone terhubung ke jaringan Wi-Fi lokal yang sama dengan server.

2

Pilih Hubungkan ke Server (Connect to Server)

Buka tab "Telusuri" (Browse), lalu ketuk ikon tiga titik (...) di sudut kanan atas layar iPhone Anda, dan pilih "Hubungkan ke Server" (Connect to Server).

3

Masukkan Alamat Server WebDAV

Masukkan alamat berikut ke dalam kolom server lalu ketuk Hubungkan:

http://[IP_SERVER]:5005
4

Pilih User Terdaftar & Masuk

Pilih mode "Pengguna Terdaftar" (Registered User), masukkan Username dan Password yang telah Anda konfigurasikan di panel sebelah kiri, lalu selesaikan koneksi.
